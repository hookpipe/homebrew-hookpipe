# homebrew-hookpipe

Homebrew tap for [hookpipe](https://github.com/hookpipe/hookpipe) — open-source webhook infrastructure.

## Install

```bash
brew install hookpipe/hookpipe/hookpipe
```

Or:

```bash
brew tap hookpipe/hookpipe
brew install hookpipe
```

## Update

```bash
brew upgrade hookpipe
```

## About

hookpipe is a free, open-source webhook infrastructure that deploys to Cloudflare Workers in 30 seconds. It receives webhooks from services like Stripe, GitHub, and Slack, and reliably forwards them to your API with automatic retries, signature verification, and delivery logs.

For more information, see the [hookpipe repository](https://github.com/hookpipe/hookpipe).
